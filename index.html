AI 학습의 3단계
어떻게 이렇게 똑똑해졌나?
  • 1
    사전 학습 (Pre-Training)인터넷의 수천억 개 문장을 학습. "다음 단어 맞추기" 훈련을 수주~수개월 반복
  • 2
    미세 조정 (Fine-Tuning)질문-답변 형식의 고품질 데이터로 대화 능력 강화 (수일~수주)
  • 3
    인간 피드백 학습 (RLHF)사람이 AI 답변을 평가 → 유용하고 안전한 응답을 학습. 지속적으로 진행
⚠️
RLHF의 부작용: 할루시네이션사람 평가자가 자신감 있는 답변을 선호 → AI가 "모른다"고 말하기보다 자신있게 틀린 답을 말하는 경향이 생김. 이것이 '할루시네이션'의 원인 중 하나!
5단계 작동 과정
AI는 이렇게 대답한다
1
입력 받기
사용자의 질문이나 명령을 받습니다
2
토큰화 (Tokenization)
문장을 작은 단위(토큰)로 나눕니다. "안녕하세요" → [안녕, 하세, 요] 처럼 쪼갬
→ 2회차에서 자세히 배울 내용
3
의미 공간으로 변환 (임베딩)
각 토큰을 숫자 벡터로 바꿉니다. "사랑" → [0.2, 0.8, 0.5, ...]
→ 3회차에서 자세히 배울 내용
4
어텐션 적용 ⭐ 핵심!
중요한 부분에 집중하며 처리합니다. "그것"이 무엇을 가리키는지 파악
→ 4회차에서 집중 학습 예정
5
확률적 생성
다음 단어를 확률로 선택하여 답변 생성. 이 과정이 초당 수백 번 반복됨
→ 5회차에서 자세히 배울 내용
비유로 이해하기
AI는 어떤 학생인가?
  • 📚
    규칙 기반 = 교과서 암기문법 규칙 100개를 외운 학생. 규칙에 없으면 처리 못 함 ❌
  • 🧠
    학습 기반 = 많이 듣고 말하기영어 대화를 1만 시간 들은 학생. 패턴을 자연스럽게 익혀 새 문장도 만들 수 있음 ✅
  • 🎯
    LLM = 수십억 문장을 "들은" 학생규칙을 몰라도 자연스럽게 말합니다. 단, 자신감 있게 틀릴 수도 있어요
AI가 할 수 있는 것
이해 · 생성 · 추론
  • 🧠
    이해 (Understanding)질문 의도 파악, 맥락 이해, 여러 문장의 관계 파악
  • ✍️
    생성 (Generation)새로운 텍스트 생성, 번역, 요약, 확장, 창작
  • 🔗
    추론 (Reasoning)논리적 연결, 단계적 사고, 결론 도출
⚠️
주의: "이해"라는 단어를 조심스럽게 사용하세요AI가 정말 이해하는 것인지, 아니면 복잡한 패턴 매칭인지 — 이것은 6회차의 핵심 주제입니다!
